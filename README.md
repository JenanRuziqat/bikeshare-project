### Project Title
US Bikeshare Data Analysis

### Description
This Python script analyzes bikeshare usage data from three US cities: Chicago, New York City, and Washington. Users can interactively filter the data by city, month, and day of week to view statistics about trip times, popular stations, trip duration, and user demographics.

### Files used
- bikeshare.py - Main Python script for data analysis
- chicago.csv, new_york_city.csv, washington.csv - Data files (not tracked in repository)
- .gitignore - Excludes CSV data files from version control
